页面布局方案

工具面板如何处理各功能区的位置关系

A

统一深色面板（推荐）

整页深色 coding-surface 风格，各功能区以卡片形式嵌入同一个深色背景中，像 VS Code 或终端工具

B

奶油顶栏 + 深色工具区

顶部奶油色品牌 header（衬线标题），下方深色操作区 —— 保留品牌氛围但主体仍是工具感

C

侧栏布局

左侧深色设置面板（质量、选项固定可见），右侧主内容区（拖拽区 + 文件列表）。更像桌面应用
